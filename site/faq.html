Frequently Asked Questions


How does this service work?

It works by monitoring the track ID you provide on Spotify's Web API (and using Track Relinking if the ID you provided is still not available).

What is "Spotify ID of the song"?

It's the copied big string after the spotify:track: when you right-click on the song -> Share -> URI

How can I delete my checkers?

You can use Delete checker page. Be aware that if you don't enter the song ID, every checker associated with that mail address will be deleted.
